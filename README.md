## Projects

### Vakya
**AI-powered legal contract analysis platform**

Vakya is built to analyze contracts, extract clauses, detect risks, summarize legal documents, and provide intelligent contract assistance.

**Tech used:**
- Frontend: React, TypeScript, Vite
- Backend: Python, FastAPI
- Database: PostgreSQL
- AI Orchestration: LangChain, LangGraph
- Local LLM: Ollama (`qwen3:8b`)
- Vector DB: ChromaDB
- Embeddings: Sentence-Transformers
- Document Processing: PyMuPDF, python-docx, PaddleOCR, Tesseract OCR, spaCy, langdetect
- Authentication: Google OAuth
- Deployment / DevOps: Docker

### Scholar-Judge
**Secure coding platform with AI mentorship**

Scholar-Judge is a hybrid coding platform with a secure Docker-based judge and an AI mentorship layer that gives Socratic hints for failed submissions.

**Tech used:**
- Frontend: React, Next.js
- Backend: Django / FastAPI
- Async / Queue: Celery, Redis
- Database: PostgreSQL
- Sandbox / Judge: Docker
- AI Integration: LLM-based hint system
